# 🚀 نظرة عامة

**𖠌** هو بوت تيليجرام يعتمد على مكتبة **python-telegram-bot** ويعمل بنظام **Polling** دون الحاجة إلى Webhook، ويوفر نظاماً متكاملاً لإدارة رسائل الترحيب والأزرار الشفافة داخل القنوات.

---

# ✨ المميزات

- 🎨 رسالة ترحيبية مخصصة لكل قناة.
- 🖼️ إرسال صورة مع رسالة الترحيب.
- 🔘 دعم الأزرار الشفافة (Inline Buttons).
- 📢 إدارة مستقلة لكل قناة.
- ⚡ سرعة واستقرار في الأداء.
- 🛡️ تخزين البيانات بشكل منظم.
- ☁️ جاهز للعمل على Railway.
- 🔄 تشغيل مستمر دون الحاجة إلى Webhook.

# ☁️ التشغيل على Railway

1. ارفع المشروع إلى GitHub.
2. اربطه مع Railway.
3. أضف المتغير البيئي:

```text
BOT_TOKEN=توكن_البوت
```

4. واجعل أمر التشغيل:

```bash
python bot.py
```
