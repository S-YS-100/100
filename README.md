# 🤖 بوت شاهين لصنع الأزرار الشفافة والترحيب المخصص | SHAHEEN YS Bot 2026 𖠌

بوت تليجرام متطور ومبني بالكامل بلغة **Python** لعام 2026، مصمم لمساعدة مدراء القنوات والبوتات في توليد لستات أزرار شفافة مخصصة (`InlineKeyboardMarkup`) وإدارة رسائل الترحيب التفاعلية بشكل مستقل لكل قناة. يعمل البوت بنظام الـ **Polling** المستقر دون الحاجة لتعقيدات الـ Webhook.

---

## ✨ الميزات الأساسية (Features)

* **هوية مخصصة 𖠌:** البوت يحمل اسم `𝚂𝙷𝙰𝙷𝙴𝙴𝙽 | 𝚈𝚂` مع صورة متحركة (GIF) ترحيبية تنادي المستخدم باسمه تلقائياً عند النقر على `/start`.
* **إضافة مرنة للقنوات:** يدعم إضافة القنوات والبوتات بـ 3 طرق (معرف القناة `@`, الـ ID الرقمي، أو عبر توجيه/تحويل رسالة من القناة مباشرة).
* **أزرار تصفح مخصصة:** إمكانية ربط أي رابط خارجي، يوزر قناة، أو يوزر بوت داخل الأزرار الشفافة مع تسمية مخصصة لكل زر.
* **ترحيب مخصص لكل قناة:** ميزة فحص انضمام الأعضاء الجدد للقنوات المضافة، وإرسال رسالة ترحيبية خاصة تم تعيينها مسبقاً لكل قناة على حدة.
* **قسم مطور متكامل 👑:** يحتوي على زر شفاف خاص يعرض بطاقة المطور التعريفية وشبكة من 6 أزرار شفافة متناسقة للوصول إلى حسابات التواصل الاجتماعي الخاصة بالمطور **يوسف شاهين**.
* **قاعدة بيانات مدمجة:** يعتمد على قاعدة بيانات `SQLite` محلية تنشأ ذاتياً وتخزن البيانات بأمان دون الحاجة لإعدادات معقدة.

## 🛠️ متطلبات التشغيل (Requirements)

* **بيئة التشغيل:** Python 3.10 أو أحدث.
* **المكتبات المطلوبة:** `python-telegram-bot` إصدار 21 فما فوق (مكتوبة داخل ملف `requirements.txt`).

---

## 🚀 طريقة التنصيب والتشغيل المحلي (Local Deployment)

1. قم بتحميل أو استنساخ المستودع:
   ```bash
   git clone https://github.com
   cd SHAHEEN-YS-Bot
   ```

2. تثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```

3. تعيين التوكن وتشغيل البوت:
   * قم بتعيين متغير البيئة `BOT_TOKEN` في جهازك، أو ضع التوكن مباشرة داخل ملف `bot.py` في حقل `TOKEN`.
   * شغل البوت:
   ```bash
   python bot.py
   ```

---

## ☁️ الرفع على منصة Railway (Railway Deployment)

1. ارفع المشروع إلى مستودع خاص بك على GitHub.
2. اربط المستودع بمنصة **Railway**.
3. اذهب إلى تبويب **Variables** وأضف المتغير التالي:
   * **Key:** `BOT_TOKEN` | **Value:** (توكن البوت الخاص بك من BotFather)
4. اذهب إلى تبويب **Settings** وضع أمر التشغيل (Start Command):
   ```bash
   python bot.py
   ```
